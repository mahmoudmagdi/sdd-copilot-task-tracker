# SDD Copilot Task Tracker

This is a demo project showing how to use Spec-Driven Development with GitHub Copilot.

## Demo Goal

Show how a team can move from:

Spec -> GitHub Issue -> Copilot Implementation -> Pull Request -> Review -> Merge

## Tech Stack

- Next.js

- TypeScript

- Tailwind CSS

- Prisma

- SQLite

- Zod

## Specs

See the `/specs` directory.

## Local Development

```bash

npm install

npm run db:generate

npm run db:migrate

npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
